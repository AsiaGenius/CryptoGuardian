========
Install CryptoGuardian 
========

Step #1 Hexed
====

Primeiramente, deverá anexar a ring.dll em seu hexed. Vá até o ``https://github.com/AsiaGenius/CryptoGuardian`` e na 
pasta ``Tools``, abra o ``LordPE``.

|step1|

Clique em PE Editor e escolha o hexed
